# STEP1
1. 변수생성 
- 컴퓨터가 제시할 임임의 정수 3개를 담아둘 변수
- 남은 시도횟수를 담아둘 변수 (초기값 9)
2. 함수생성
- 1~9 사이의 세 개의 임의의 정수 생성 후 반환하는 함수 (컴퓨터가 생성한 임의의 정수)
```pcRandom()```

- 1~9 사이의 세 개의 임의의 정수 생성 후 반환하는 함수 (사용자가 생성한 임의의 정수)
```userRandom()```

- 컴퓨터와 사용자가 생성한 임의의 정수를 비교하여 볼과 스트라이크 결과를 반환하는 함수
```comparisonStrikBall()```
